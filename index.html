Implementation notes

Hooks are installed via Dobby at runtime. Each hook uses a trampoline to preserve original behaviour where needed. The REDIRECT_FLAG coordinates between DNS spoofing and connect-time redirection.

// gethostbyname hook — redirect blocked domains to 127.0.0.1
struct hostent* hookedGetHostByName(const char *name) {
    if (name && is_blocked(name)) {
        static struct hostent redirected_host;
        // fill with loopback 127.0.0.1
        REDIRECT_FLAG = 1;
        return &redirected_host;
    }
    REDIRECT_FLAG = 0;
    return originalGetHostByName(name);
}
